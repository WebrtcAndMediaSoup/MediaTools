### 视频文件分离音视频

```shell
ffmpeg -i input.mp4 -map 0:v -c:v copy video.264 -map 0:a -c:a copy audio.aac
```



### MP4 文件转换为 FLV 文件

```
ffmpeg -i input.mp4 -c:v libx264 -c:a aac output.flv
```

#### **媒体测试集成demo**

src目录下simplest_mediadata_test，目前已转成vs2019编译

本项目包含如下几种视音频数据解析示例：
 (1)像素数据处理程序。包含RGB和YUV像素格式处理的函数。
 (2)音频采样数据处理程序。包含PCM音频采样格式处理的函数。
 (3)H.264码流分析程序。可以分离并解析NALU。
 (4)AAC码流分析程序。可以分离并解析ADTS帧。
 (5)FLV封装格式分析程序。可以将FLV中的MP3音频码流分离出来。
 (6)UDP-RTP协议分析程序。可以将分析UDP/RTP/MPEG-TS数据包。
相关介绍：https://blog.csdn.net/leixiaohua1020/article/details/50535042
